Welcome to ios blackjack tutorial github page

In this github repository I've created an iPhone app in XCode. The app that can be built from this sourcecode will run on a iOS 6 device, and in more specifically I've created it for a iPhone 5. Meaning I've only made sure it looks ok on the iPhone 5 (Simulator).

This code is the result of this IOS blackjack tutorial created by me.

The project is created with the following techniques and design decisions:

  • XCode
  • Objective-C
  • Model View Controller
  • Key Value Observing (automatic and manual)

If you build the sources in Xcode and you will run it in the iphone simulator it will result in a working blackjack:
